## Описание

Клиентская часть чата с использованием WebSocket. Реализован вход по никнейму с проверкой занятости, отображение списка пользователей онлайн, отправка и получение сообщений в реальном времени.

## Технологии

- Webpack
- WebSocket (ws)
- GitHub Pages + Actions
- Render.com (бэкенд)

## Локальный запуск

```bash
yarn install
yarn dev
